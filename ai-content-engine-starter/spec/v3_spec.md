# V3 Spec

## Scope

V3 introduces an editorial intelligence layer on top of the V2 pipeline.

New modules:

- editorial planner
- content assets
- story clusters
- adaptive ranking
- network coordination
- monetization
- policy engine
- operations layer

## Constraints

- single Go service
- modular packages
- feature flags controlling rollout
- no microservices
- backward compatible with V2 pipeline
